# Growlity Academy Presentation

---

## Slide 1: Project Overview & Tech Stack

### **Project: Growlity Academy**
Growlity Academy is a comprehensive e-learning ecosystem designed for individual learners, corporate teams (B2B), and instructors. It provides a seamless experience for course discovery, interactive learning, and certification.

### **Core Technology Stack**
- **Frontend**: **Next.js** (Utilizing App Router & SSR for performance)
- **Backend**: **Node.js** with **Express**
- **Database**: **PostgreSQL** (Managed via **Prisma ORM**)
- **Storage**: **AWS S3** (For course videos and assets)
- **Caching**: **Redis** (For optimized performance and session handling)

---

## Slide 2: Key Features Implemented

### **Foundational Development**
- **End-to-End Setup**: Orchestrated the integration between the Next.js frontend and Express backend.
- **Secure Authentication**: Implemented JWT-based login and signup systems with secure session handling.

### **Administrative Ecosystem**
- **Category Management**: Developed a dynamic CRUD system (Backend & Frontend) for organizing course catalogs.
- **Subscription Engine**: Built the infrastructure for managing recurring subscription plans and user access levels.
- **Admin Dashboard UI**: Designed a premium, responsive dashboard layout with a focus on usability.

### **Optimization & Integration**
- **Mobile-First Design**: Ensured the administrative interface is fully functional on mobile and tablet devices.
- **Advanced Data Flow**: Connected Prisma-backed APIs with the frontend to ensure real-time data accuracy.
